Within Python there is a function which allows you to iterate and complete operations n number of times this is called the for loop
There is also the while loop which is used to run continually when a specific condition is met but I will talk about that later.

Example 
for i in range(10):
   print i
Above is an incredibly simple example of how a for loop will be used. I will use some more complicated below to help your understanding.

Explanation
  1. The for keyword is used to iterate over numbers from 0 to 9. Range is a built in Python function which is used to print out a specific amount of numbers between a given point.

    Range has a start, stop and end.
    • Start: Start range from this number
    • Stop: Produce numbers up to, but not including the stop number. By default range uses stop if only one number is in parantheses
    • Step: Defines how many steps that is to be taken during number generation i.e.
    for i in range(start,stop,step)

Below is an example of using the range function in a different way, here I start at the number 2, end at but do not include 12 and generate the numbers in a step sequence of 2.

for j in range(2,12,2):
   print j
