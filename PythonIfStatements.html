The if statement is a conditional statement which is based around whether something is either true or false. An if statement can work alongside the elif and else statements to offer a more detailed program.

Elif means if the initial 'if' statement is not true then run the code inside the elif statement.


Else means run this if none of the if or elif statements are true.

Example 
b = 5
if b==5:
   print("b is equal to 5")
elif b==10:
   print("b is equal to 10")
else:
   print("None of the if or elif are true")
Outcome 
"b is equal to 5"
Explanation

  1. Ok so we have a variable called b which has a value of 5.

  2. With the if statement we are asking if b is equal to 5 which it is at that moment, so the condition is true so the code inside that condition is run.

  3. We also have an elif statement which will be run if b is equal to 10.

  4. Finally, we have an else statement which will run if no other condition is true

Nested If statements

From time to time within a program you may need to test more than expression to reach a conclusion. This is where nested if statements come in handy.

With nested if statements the outer if statement has to be true for the statement to print out the inner if statement.
Example 
a = "deer"
b = "dear"
c = "debra"
if a=="deer":
   print "Hello deer"
   if b=="dear":
      print "Hello dear"
      if c=="debra":
         print "Hello debra"
Outcome 
"Hello deer"
"Hello dear"
"Hello debra"

Explanation
  1. As you can see from the example all conditions are true so each code segment inside each if statement gets printed.

  2. However, if we were to change the initial value of b and change it to deck you will see that the output will differ and only one result will be shown because the condition is not true and will not print itself nor its inner statement which is dear and debra. See below for more detail.
Explanation two 
a = "deer"
b = "deck" #b has been changed to deck
c = "debra"
if a=="deer":
   print "Hello deer"
   if b=="dear":
      print "Hello dear"
      if c=="debra":
         print "Hello debra"
Outcome 
"Hello deer" #deer only gets printed as value of b
 #has changed so neither b nor c gets printed
