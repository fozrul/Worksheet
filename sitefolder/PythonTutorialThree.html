When creating large programs it is vital if you are creating individually or within a group that you know what each bit of code does and whether it takes results from a module or whether an external library is required as this is imperative for collaboration.

To make a single line comment in Python simply type the '#'

However to create a multiline comment type """ for the start of the comment and then type """ again to signify the end of the comment
