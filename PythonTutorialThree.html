To make a single line comment in Python simply type the '#'

However to create a multiline comment type """ for the start of the comment and then type """ again to signify the end of the comment
For future's sake I will be using Python 3 for these tutorials, specifically Python 3.4. To print strings within Python you enclose statements either with a single or double quotation mark, both will achieve the desired result.
